This repository is an Infrastructure-as-Code (Terraform) demo for Alibaba Cloud Landing Zone Accelerator. It contains zero frontend assets: no CSS/SCSS/Sass/Less/Stylus files, no HTML templates, no JavaScript/TypeScript UI code, and no references to any frontend framework or styling library (Tailwind, Bootstrap, MUI, Ant Design, etc.). The entire codebase consists of Terraform modules, GitHub Actions workflows, and configuration YAML/JSON files used to provision cloud infrastructure. Consequently, the `frontend_style` category does not apply.
